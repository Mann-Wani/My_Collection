# My_Collection
This repository includes all codes, all projects, all experiments done by me using various softwares and/or on different microcontrollers. 
It includes codes for - 

8051 - Using Keil C51 and Proteus. Codes in C and Assembly
ESP32 - Using Arduino-IDE and ESP-IDF. Codes in C and C++ for Esp32-Dev-Kit
STM32 - Using STM32CubeIDE & CubeMX. Codes in C and bare-metal for STM32-Blue-Pill
Arduino - Using Arduino IDE. Codes in C and C++ for Arduino Uno 
Bash - Codes for Ubuntu terminal. Generic codes for daily tasks
Docker - Codes for automating tasks, to share projects and dependencies
